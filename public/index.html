NC News API

An API that provides access to NC News articles, topics, users and comments.

Available Endpoints:

  • GET /api/topics

    Retrieve a list of all topics.

  • GET /api/articles/:article_id

    Retrieve details of a specific article by its ID.

  • GET /api/articles/:article_id/comments

    Retrieve all comments for a specific article.

  • GET /api/comments/:comment_id

    Retrieve details of a specific comment.

  • POST /api/comments

    Add a new comment to an article. Requires a request body with 'username' and 'body'.

  • GET /api/articles

    Retrieve a list of all articles. Optional query parameters: topic, sort_by, order.

  • GET /api/users

    Retrieve a list of all users.

  • GET /api/topics

    Retrieve all topics available in the database.
